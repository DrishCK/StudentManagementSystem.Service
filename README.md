# Student Management System

## Overview
The Student Management System is a comprehensive solution designed to help educational institutions manage student information efficiently. It provides functionalities for tracking student enrollment, performance, and personal details.

## Features
- **Student Registration**: Allows new students to register with their personal details.
- **Course Enrollment**: Students can enroll in various courses offered by the institution.
- **Grade Management**: Educators can enter and manage grades for each student for different courses.
- **Reporting**: Generate reports on student performance, attendance, and overall statistics.
- **User Management**: Admins can manage users, including students and faculty.

## Technologies Used
- **Backend**: ASP.NET Core
- **Database**: SQL Server
- **Frontend**: Angular
